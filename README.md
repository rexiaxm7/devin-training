# devin-training

## ○×ゲーム（Tic-Tac-Toe）

Golangで実装したコンソール上で動作する○×ゲームです。

### 実行方法

```bash
go run cmd/tictactoe/main.go
```

### ゲームルール

- 3×3のマス目を使用します
- プレイヤーは2人（「○」と「×」）で交互に手を打ちます
- 縦・横・斜めのいずれかに3つ並んだら勝利です
- すべてのマスが埋まり、勝敗がつかなかった場合は引き分けです

### 入力方法

- 0〜8の番号を入力して、対応するマスに記号を配置します
- マス目の番号は以下のように配置されています：
  ```
  0 | 1 | 2
  ---------
  3 | 4 | 5
  ---------
  6 | 7 | 8
  ```
